① Populate our standardized Excel matrix template with your structured scripts, voiceover tracks, and image prompt arguments generated from your primary LLMs.

④ Open CapCut to immediately apply localized transitions, background music layers, and branding presets before exporting your final video file.

Mandatory Structural Rules ★ Compulsory Spec

To allow the image engine and localized acoustic transcription tasks to loop across data sheets without pathing failures, your spreadsheet layout must strictly comply with the parameters below.
Configure your baseline text prompt formatting so that inputs copy and paste smoothly into these exact matrix slots.
📥 Pipeline Inputs
Raw text outputs generated via standard LLM prompt files
📤 Expected Outputs
Perfectly structured Excel template file (.xlsx)
[Excel Layout Map Example]
Sheet 1 Title: "Script" → Column Header Required: "Image Prompt"
Sheet 2 Title: "Voiceover" → Column Header Required: "Voiceover Script"
▲ MS Excel Data Scheme Configuration Framework Verification (Fig.1)
Component Specifications
1
Core Script Sheet Matrix
  • The primary computational sheet holding your narrative arc and asset assignments.
  • You can construct custom administrative variables across custom surrounding columns.
  • However, to feed the batch rendering engine, you must contain a column titled exactly "Image Prompt".
2
"Image Prompt" Character Header Definition ★ Required
  • The automation engine queries this exact column string parameter to feed background rendering loops.
  • Column grid placement is flexible, but the key phrase string must match exactly: "Image Prompt".
3
Voiceover Data Sheet Configuration ★ Required
  • The core vocal generator parses this target layer to process language blocks and output audio formats.
  • The data sheet name container must match exactly: "Voiceover".
  • The processing header holding text blocks for acoustic rendering must be explicitly titled: "Voiceover Script".
  • Refer to the reference file: AI_Voiceover_Script_Generation_Prompt.txt.
System Verification Requirements
The data sheet title identifier must match exactly: "Voiceover" — case parameters, character spacing, and syntax are checked strictly.
The processing column header must match exactly: "Image Prompt" — alternate names will break the automated parsing loops.
Failing to align these key naming tokens will cause media asset file loops to throw unexpected execution faults.

Technical Concept Overview — Audio Pipeline Mechanics
🔊
What is the underlying TTS architecture?
Text-to-Speech (TTS) technology converts written data parameters into synthesized natural voice frequencies. AutoTube Pro implements specialized Qwen3-TTS weights to match the exact pitch, intonation, and tone of your target vocal audio sample reference files.
☁️
Why leverage Google Colab server structures?
Processing high-fidelity voice cloning models require substantial GPU computing power. Utilizing Google Colab lets you host the model on a remote cloud server for free, avoiding high local hardware resource usage while maximizing speech synthesis speeds.
🔗
System Connection Lifecycle Mapping
Launching your Colab script generates a secure, unique live tunnel link (`gradio.live`). AutoTube Pro uses this link to send script strings directly to the cloud model, which processes the speech synthesis task and returns clean audio streams back to your PC instantly.
AutoTube Pro Client → Transmits Text Blocks → Remote Colab Server Cluster (Compute Audio) → Returns Streamed Audio Files Logged Locally

Operational Controls Description
1
Configure Source Data Sheet Document & Consolidated Audio Track Paths
  • Source Sheet Route: Map the path to the active project spreadsheet located inside your 1_Script\ path.
  • Master Track Route: Point this path argument directly to the consolidated master.wav file inside the 3_Audio_Subtitles\ directory.
2

  • Phase 1 — Generate Master Caption SRT: Launches localized Whisper AI tools to read the master audio track and export standard SRT files.

Skipping this step will break the time layout model, causing **CapCut to map image layers onto mismatched audio tracks**.
Logging problematic rows inside the flag field simplifies your final asset quality review checks in Step 6.
