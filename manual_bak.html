폴더 구조 설정
AutoTube Pro의 모든 자동화 기능은 정해진 폴더 구조를 기반으로 동작합니다.
작업을 시작하기 전에 아래 순서대로 폴더를 설정하세요.
설치 순서
1
제공된 패키지 파일 다운로드
  • AutoTube Pro 구매 후 제공되는 패키지 파일(.zip)을 다운로드합니다.
2
C: 드라이브에 압축 해제 ★ 위치 필수
  • 다운로드한 파일을 반드시 C: 드라이브 최상단에 압축 해제합니다.
  • 압축 해제 후 C:\유튜브_영상제작\ 폴더가 생성됩니다.
3
새 영상 작업 시 프로젝트 폴더 추가
  • 영상_작업파일\ 안에 새 프로젝트명으로 폴더를 만들고 하위 3개 폴더를 그대로 복사합니다.
  • 예: 어닝시즌에살아남기\ → 금리인하의비밀\
폴더 구조
💻 C:\
📁 유튜브_영상제작\ ← 반드시 C: 드라이브에 위치
📁 영상_작업파일\
📁 어닝시즌에살아남기\ ← 프로젝트명 (자유롭게 설정)
📂 1_대본\ STEP 1 · 2 엑셀 대본 파일 보관
📂 2_이미지\ STEP 3 자동 생성 이미지 저장
📂 3_오디오_자막\ STEP 4 · 5 음성·자막 파일 저장
폴더관련 STEP자동 저장되는 파일
📂 1_대본\STEP 1STEP 2엑셀 대본 파일 (.xlsx)
📂 2_이미지\STEP 301.png, 02.png, 03.png …
📂 3_오디오_자막\STEP 4STEP 5001.wav … 합본.wav, 자막.srt
핵심설명
C: 드라이브가 아닌 다른 위치에 설치하면 자동화가 동작하지 않습니다.
새 영상 작업 시 프로젝트 폴더(예: 어닝시즌에살아남기)만 새로 만들면 됩니다. 하위 3개 폴더 구조는 동일하게 유지합니다.
각 STEP에서 경로를 지정할 때 해당 폴더를 직접 선택하면 됩니다.
STEP 1

엑셀 파일 필수 구조 규칙 ★ 절대 필수
이미지 자동 생성과 TTS 오디오 녹음이 정상 동작하려면,
아래 규칙을 반드시 준수하여 엑셀 파일을 만들어야 합니다.
기존 프롬프트로 대본을 생성하실 때 엑셀에 복사·붙여넣기가 가능한 형태로 요청하시면 됩니다.

각 번호별 기능 설명
1
대본 시트 — 메인 대본 내용 구성
  • 영상의 메인 대본 내용이 담긴 기본 시트입니다.
  • 사용자가 관리하는 내용으로 자유롭게 구성하시면 됩니다.
  • 단, 자동 이미지 생성을 위해 "이미지 생성 프롬프트" 열은 반드시 포함되어야 합니다. ★
2
"이미지 생성 프롬프트" 열 제목 ★ 필수
  • 이미지 자동 생성 시 프로그램이 이 열에서 프롬프트를 읽습니다.
  • 열의 위치는 어디든 상관없지만, 열 제목은 정확히 "이미지 생성 프롬프트"여야 합니다.
3
성우녹음 시트 ★ 필수
  • TTS 오디오 생성 시 이 시트의 텍스트를 읽어 음성을 생성합니다.
  • 시트 이름은 반드시 "성우녹음" 4글자여야 합니다.
  • TTS 오디오를 생성할 열 이름은 반드시 "성우녹음대본"으로 설정하세요.
  • 제공되는 AI성우_녹음대본_생성프롬프트.txt 파일을 참고하세요.
핵심설명
시트 이름은 반드시 "성우녹음" — 글자 수·띄어쓰기 모두 정확해야 합니다.
열 제목은 반드시 "이미지 생성 프롬프트" — 유사 표현은 인식 불가입니다.
위 조건 미충족 시 이미지·오디오 생성이 전혀 동작하지 않습니다.

AI로 생성된 대본 내용 (복사 완료)

1
대본/기획 메인 탭

2
프롬프트 폴더 경로 입력
  • 찾기 버튼으로 1_대본\ 폴더를 선택합니다.

2
자동 정렬할 열 선택 드롭다운
  • 열 너비와 행 높이를 기준으로 맞출 열(column)을 선택합니다. (예: 성우녹음대본)
3
↔ 자동 맞춤 버튼
  • 선택한 열의 텍스트 길이에 맞게 열 너비·행 높이가 자동 최적화됩니다.
4
🔢 번호 정렬 버튼
  • 행 추가·삭제 후 어긋난 순번을 1부터 재정렬합니다.
  • 단, 첫 번째 열 이름에 '번호' 또는 'no'가 포함된 경우에만 작동합니다.

하단 시트 탭 (대본 / 성우녹음)

  • 성우녹음 시트: TTS 녹음의 필수 시트입니다. (→ STEP 1 규칙 참고)

각 번호별 기능 설명
1
작업 파일 경로 & 캐릭터 이미지 경로
  • 작업 파일 경로: 이미지를 생성할 엑셀 대본 파일을 선택합니다. (1_대본\ 폴더 안의 .xlsx 파일)
  • 캐릭터 이미지: 영상 전체 화풍과 캐릭터를 통일하기 위한 기준 이미지(.png)입니다. AI가 이 이미지를 참고해 일관된 화풍으로 생성합니다.
2
생성 범위 설정 & 끝번호 확인
  • 엑셀 파일 지정 시 시작~끝 번호가 자동으로 채워집니다.
  • 개별생성: 특정 이미지만 다시 만들 때 번호를 직접 입력합니다.
3
구글 로그인 / 이미지 자동 생성 시작 / 작업 중지
  • 구글 로그인: Gemini AI는 Google 계정 기반으로 동작하므로 작업 전 1회 로그인이 필요합니다. 이후 자동 유지됩니다.
  • 이미지 자동 생성 시작: 버튼 클릭 후 자리를 비워도 자동으로 진행됩니다.
  • 완성 이미지는 2_이미지\ 폴더에 01.png, 02.png 형식으로 자동 저장됩니다.
4
진행상태 패널
  • 전체 / 현재 진행 / 남은 작업 / 진행률(%)을 실시간 표시합니다.
5
실시간 이미지 미리보기
  • 생성 완료된 이미지가 하단에 즉시 표시됩니다. 품질을 실시간으로 확인하세요.
핵심설명
구글 로그인은 최초 1회만 필요합니다. 이후 자동 유지됩니다.
이미지 생성 중에 STEP 4 오디오 생성을 동시에 진행하면 전체 시간을 단축할 수 있습니다. 단, PC 성능에 따라 동시 작업이 어려울 수 있습니다.
생성된 이미지는 2_이미지\ 폴더에 자동 저장됩니다. 별도 파일명 변경이 필요 없습니다.

STEP 4 시작 전 — 기본 개념 이해
🔊
TTS란 무엇인가요?
TTS(Text-to-Speech)는 텍스트를 음성으로 변환하는 기술입니다. AutoTube Pro는 Qwen3-TTS 모델을 사용하여 제공하신 목소리 샘플과 거의 동일한 억양·톤으로 대본을 읽어줍니다.
☁️
왜 Google Colab을 사용하나요?
Qwen3-TTS 모델은 고품질 음성을 만들기 위해 강력한 컴퓨팅 자원이 필요합니다. Google Colab은 구글이 무료로 제공하는 클라우드 서버입니다. 내 PC 성능에 부담을 주지 않으면서 고품질 TTS를 무료로 사용할 수 있습니다.
🔗
전체 동작 방식
Colab에서 TTS 서버를 실행하면 gradio.live 웹 주소가 생성됩니다. AutoTube Pro가 이 주소로 대본 텍스트를 전송하면 Colab 서버가 음성을 생성해서 돌려보냅니다. 생성된 TTS 음성은 자동으로 내 PC에 저장 됩니다.
AutoTube Pro → 대본 텍스트 전송 → Colab 서버 (음성 생성) → AutoTube Pro 저장

① AutoTube Pro — 오디오 준비 설정
오디오 생성 전 AutoTube Pro에서 먼저 필요한 정보를 설정하는 단계입니다.
목소리 샘플 파일과 복사할 내용을 지정하고, Colab 접속에 필요한 정보를 복사합니다.
📥 이 단계를 시작하기 전에
목소리 샘플 .wav 파일 (5~15초)
📤 이 단계가 끝나면
Colab URL + 명령코드 복사 완료
▲ [2] 오디오 자동 생성 패널 — 상단 설정 영역 (그림5)
💡 이 단계에서는 화면 상단 영역(배지 ①②③)만 설정합니다. 하단 녹음 버튼은 STEP 4-③에서 사용합니다.
각 번호별 기능 설명
1
목소리 원본 & 복사할 내용
  • 목소리 원본(.wav): AI가 억양·톤 스타일을 참고할 기준 음성 파일입니다. 5~15초 분량의 깨끗한 목소리 샘플을 권장합니다.
  • 복사할 내용: 해당 음성 파일 안에서 성우가 실제로 말하는 텍스트를 그대로 입력합니다. AI가 목소리와 텍스트를 대응시켜 억양 패턴을 학습합니다.
2
1) 주소복사 버튼
  • Google Colab 노트북 URL을 클립보드에 복사합니다.
  • 복사 후 브라우저 주소창에 붙여넣어 Colab 노트북을 엽니다.

2) 명령코드 / 3) 명령코드 버튼

  • STEP 4-②에서 Colab 화면에 순서대로 붙여넣어 실행합니다.
② Google Colab — TTS 서버 가동
Google Colab은 구글이 제공하는 무료 클라우드 컴퓨팅 환경입니다.
내 PC 성능에 영향을 주지 않고 TTS(음성 합성) 서버를 무료로 가동할 수 있습니다.
배지 번호 ④ → ⑤ → ⑥ → ⑦ 순서로 진행합니다.
📥 이 단계를 시작하기 전에
AutoTube Pro에서 "코랩 열기" 버튼 클릭
📤 이 단계가 끝나면
gradio.live TTS 서버 주소 발급 및 복사 완료

4
AutoTube Pro "코랩 열기" 버튼 클릭
  • AutoTube Pro의 "코랩 열기" 버튼을 클릭하면 브라우저에서 노트북이 자동으로 열립니다.
  • 노트북을 열 때 아래와 같은 경고 메시지가 표시될 수 있습니다.
    "이 노트북은 Google에서 작성하지 않았습니다"
    이는 GitHub에서 불러올 때 구글이 표시하는 정상적인 보안 안내입니다. 무시하고 계속 진행하세요.
5
T4 GPU 설정 확인 매번 확인
  • 상단 메뉴 수정 → 노트북 설정을 클릭합니다.
  • 하드웨어 가속기가 T4 GPU로 설정되어 있는지 확인합니다.
  • T4 GPU가 아닌 경우 선택 후 저장하세요. CPU로 실행 시 속도가 매우 느립니다.
6
셀1 — 필수 패키지 설치 최초 1회
  • 첫 번째 셀의 ▶ 실행을 클릭합니다.
  • 실행 직후 바로 메시지가 출력될 수 있으나 아직 설치 중인 상태입니다. 바로 다음 셀을 실행하지 마세요.
  • 약 30초~1분 후 셀 하단에 최종 "✅ 설치가 완료되었습니다!" 메시지가 다시 출력됩니다.
  • 이 최종 완료 메시지를 반드시 확인한 후에만 셀2를 실행하세요. 너무 빨리 셀2를 누르면 오류가 발생합니다.
7
셀2 — Qwen3-TTS 서버 실행
  • 두 번째 셀의 ▶ 실행을 클릭합니다.
  • 약 2~3분 후 셀 하단에 초록색 주소창과 📋 복사 버튼이 나타납니다.
  • 복사 버튼을 클릭하면 주소가 클립보드에 복사됩니다.
  • AutoTube Pro의 TTS 서버주소 칸에 붙여넣기 하면 연결 완료입니다.
  • 이 주소는 Colab 세션 종료 시 만료됩니다. 재시작 시 셀2만 다시 실행하면 됩니다.
핵심설명
셀1 설치는 최초 1회만 실행합니다. 이후에는 셀2만 실행하면 됩니다.
셀1 완료 메시지를 반드시 확인 후 셀2를 실행하세요. 실행 직후 바로 보이는 메시지는 최종 완료가 아닐 수 있습니다.
복사 버튼으로 주소를 복사하여 AutoTube Pro TTS 서버주소 칸에 붙여넣으면 연결 완료입니다.
Colab 세션 만료 시 → 셀2만 재실행하여 새 주소를 받아 붙여넣으세요.

녹음 범위 & 개별생성 설정

  • 개별생성: 특정 컷만 재녹음할 때 번호를 직접 입력합니다. (예: 1,2,3,4,5)
11
대본 녹음 시작 / 녹음 중지
  • 대본 녹음 시작: 성우녹음 시트의 텍스트를 순서대로 읽어 개별 WAV 파일을 생성합니다.
  • 녹음 중지: 진행 중인 녹음을 중단합니다.
12
오디오 진행상태 & 파형 시각화
  • 전체·현재·남은 작업 수와 진행률(%)을 실시간 표시합니다.
  • 파형 캔버스: 앞무음(빨간)·발화(파란)·뒤무음(초록)으로 각 녹음의 품질을 확인합니다.

각 번호별 기능 설명
1
원본 대본 & 합친 녹음 경로 지정
  • 원본 대본: 1_대본\ 폴더의 엑셀 파일을 지정합니다.
  • 합친 녹음: 3_오디오_자막\ 폴더의 합본.wav를 지정합니다.
2
전체 자막 SRT 만들기 → 엑셀에 이미지 배치 타임 적용 ★ 두 버튼 모두 클릭
  • 1단계 — 전체 자막 SRT 만들기: AI가 오디오를 분석해 각 컷의 시작·끝 시간을 담은 SRT 파일을 생성합니다.
  • 2단계 — 엑셀에 이미지 배치 타임 적용: 추출된 시간 정보를 엑셀 파일의 타임 열(J열)에 자동으로 기록합니다. 이 정보가 있어야 CapCut에서 이미지가 정확한 타이밍에 배치됩니다.
  • 두 번째 버튼을 반드시 클릭해야 합니다.
3
자막 작업 진행상태 패널
  • 진행률(%)과 현재 처리 중인 자막 텍스트를 실시간으로 표시합니다.
4
자막수정필요 입력 필드
  • 발음이 불명확하거나 자막이 잘못 인식된 컷 번호를 기록합니다. (예: 1, 3, 5)
  • STEP 6 영상 검증 탭에서 해당 컷만 재작업할 때 참조됩니다.
핵심설명
SRT 만들기 후 반드시 "엑셀에 이미지 배치 타임 적용"도 클릭해야 합니다.
두 번째 버튼 미클릭 시 CapCut에서 이미지 클립 타이밍이 잘못 배치됩니다.
자막수정필요에 번호를 기록해두면 STEP 6에서 해당 컷만 정확히 재작업할 수 있습니다.

캡컷 프로젝트 자동 생성 실행 ★ 최종 단계

  • 단 1초 안에 이미지·오디오·SRT 자막 타임코드가 CapCut 프로젝트에 자동 주입됩니다.

문제 발견 시 해당 컷의 오른쪽 체크박스 클릭 → 분홍색으로 마킹 → 해당 STEP으로 돌아가 개별생성으로 재작업합니다.
